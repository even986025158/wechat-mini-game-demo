# 微信小游戏部署指南

## 📱 手机体验方案（立即可用）

### 方案1：在线网页版体验
**文件**：`mobile-index.html`
**特点**：
- 专为手机浏览器优化
- 支持触屏操作
- 完整游戏功能
- 数据自动保存

**体验方式**：
1. 将 `mobile-index.html` 部署到任何Web服务器
2. 用手机浏览器访问链接
3. 无需安装，立即体验

### 方案2：微信小游戏部署

#### 前提条件
1. **微信开发者账号**：需要企业或个人认证
2. **AppID**：申请微信小游戏AppID
3. **微信开发者工具**：下载安装

#### 部署步骤

**第一步：准备文件**
```
wechat-mini-game-demo/
├── game.js          # 主游戏逻辑
├── game.json        # 配置文件
├── player.js        # 玩家系统
├── world.js         # 世界系统
├── storage.js       # 存储系统
└── utils.js         # 工具函数
```

**第二步：微信开发者工具配置**
1. 打开微信开发者工具
2. 选择"小游戏"项目类型
3. 输入AppID（或使用测试号）
4. 导入项目文件夹

**第三步：调试与上传**
1. 在模拟器中测试功能
2. 修复可能的兼容性问题
3. 上传代码到微信平台

**第四步：发布体验版**
1. 在微信后台提交审核
2. 设置体验版（最多100人）
3. 分享体验链接给测试用户
