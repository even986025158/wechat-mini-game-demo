# 🎮 饥荒风格微信小游戏 Demo

基于《饥荒》游戏机制的微信小游戏演示版本，支持手机触屏操作和自动存档功能。

## ✨ 功能特色

### 🎯 核心玩法
- **生存系统**：饥饿值、生命值、精神值三状态管理
- **昼夜循环**：白天采集资源，夜晚生存挑战
- **资源收集**：砍树、挖矿、收集食物
- **建造系统**：篝火、工作台等基础建筑

### 📱 移动优化
- **触屏控制**：虚拟摇杆 + 触控按钮
- **响应式设计**：适配各种手机屏幕尺寸
- **手势操作**：滑动、点击等交互方式

### 💾 数据持久化
- **自动存档**：游戏进度自动保存
- **本地存储**：使用浏览器本地存储技术
- **断点续玩**：下次启动从上次进度继续

## 🚀 快速开始

### 在线体验
1. 将项目上传到 GitHub Pages、Netlify 或 Vercel
2. 用手机浏览器访问部署地址
3. 开始游戏体验

### 本地开发
```bash
# 克隆项目
git clone <repository-url>

# 启动本地服务器
python -m http.server 8000

# 浏览器访问
open http://localhost:8000/mobile-index.html
```

## 🎮 操作说明

### 桌面版操作
- **移动**：WASD 或方向键
- **交互**：空格键
- **采集**：靠近资源按交互键

### 手机版操作
- **移动**：虚拟摇杆
- **交互**：点击交互按钮
- **采集**：靠近资源点击交互

## 🔧 技术栈

- **前端**：原生 JavaScript + Canvas
- **游戏引擎**：自定义游戏循环
- **存储**：LocalStorage API
- **适配**：微信小游戏 API 兼容
